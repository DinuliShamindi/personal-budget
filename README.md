# Personal Budget App

A simple and interactive **Personal Budget** web application built with **React** and deployed using **Firebase Hosting**. Track your income, expenses, and balance with ease.  

---

## Live Demo

Check out the live app here: https://personal-budget-9ed7d.web.app 

---

## Features

- Add transactions with description and amount.  
- Categorize transactions as **Income** or **Expense** manually.  
- View total **Income**, **Expenses**, and **Balance**.  
- Delete any transaction.  
- Responsive design for desktop and mobile.  
- Uses **Sri Lankan Rupees (Rs.)** as currency.  

---

## Tech Stack

- **Frontend:** React  
- **Styling:** CSS  
- **Hosting:** Firebase Hosting
